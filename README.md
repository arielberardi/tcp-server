# HTTP Server 

## Setup & Run

Requirement: cmake and some c++ compiler (gcc/clang, etc)

1) From root folder run: cmake -S . -B build -DCMAKE_BUILD_TYPE=Release (or Debug)
2) Then ./build/http-server
